Theme Changer 🌗
[live@](https://js-project-02.vercel.app/)

A simple Theme Changer Web App built using HTML, Tailwind CSS, JavaScript, and Font Awesome.
This project allows users to switch between Dark Mode and Light Mode with a single button click.

📌 Features
Toggle between Dark Mode and Light Mode
Dynamic button icon change
Responsive design using Tailwind CSS
Simple and beginner-friendly JavaScript project
🛠️ Technologies Used
HTML5
Tailwind CSS
JavaScript (DOM Manipulation)
Font Awesome Icons

📄 HTML Code

The HTML file contains:

Heading
Toggle Button
Tailwind CSS CDN
Font Awesome CDN

⚙️ JavaScript Functionality

Sets default theme to Dark Mode
Changes:
Background color
Text color
Button text
Button icon
Uses addEventListener() for theme switching
